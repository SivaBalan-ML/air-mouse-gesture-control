# 🖱️ Air Mouse - Hand Gesture Controller

Control your PC mouse using hand gestures in the air via your webcam!

## 🚀 Features
- Move cursor with index finger
- Left click by pinching thumb and index finger
- Scroll by raising two fingers

## 🛠️ Built With
- Python 3.12
- MediaPipe (Google AI hand detection)
- OpenCV (Computer Vision)
- PyAutoGUI (Mouse control)

## ✋ Gestures

| Gesture | Action |
|---|---|
| ☝️ Index finger up | Move cursor |
| 👌 Pinch (thumb + index) | Left click |
| ✌️ Two fingers up | Scroll |
| Q key | Quit |
